# 🍽 PHP Restaurant Booking Website

A simple restaurant table booking system built with PHP and MySQL.

![Booking Form](./screenshots/booking-form.png)

## 🧩 Features

- 📝 User can book tables (name, phone, date, time, guests, note)
- 👩‍💼 Admin login to view all bookings
- 🔐 Login & Register system using `password_hash()`
- 🗃 Booking history for users
- 📧 Email confirmation (optional)

## 🛠 Built With

- PHP (plain, no framework)
- MySQL
- HTML / CSS / JavaScript
- Bootstrap (optional)
